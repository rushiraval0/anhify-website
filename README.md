# Import Export Business Website

A modern, responsive website for an import-export business built with React.js and Tailwind CSS. The website features a dynamic product catalog, company information, and contact details.

## Features

- Responsive design that works on all devices
- Dynamic product catalog with search and filtering
- Modern UI with smooth animations
- Company information and team section
- Contact information and office locations
- Social media integration

## Technologies Used

- React.js
- TypeScript
- Tailwind CSS
- React Router
- Framer Motion
- React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd your-project-name
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Project Structure

```
src/
  ├── components/     # React components
  │   ├── Navbar.tsx
  │   ├── Home.tsx
  │   ├── Products.tsx
  │   ├── About.tsx
  │   └── Footer.tsx
  ├── App.tsx        # Main application component
  ├── index.tsx      # Application entry point
  └── index.css      # Global styles and Tailwind CSS
```

## Customization

1. Update the company information in the components
2. Replace placeholder images with actual product and team member images
3. Update contact information and social media links
4. Modify the color scheme in `tailwind.config.js`
